# Minal - AI特化ターミナルエミュレータ

## コンセプト

既存のターミナル（Wezterm, Ghostty, Alacritty）は高速・高機能だが、AI時代の開発ワークフローに最適化されていない。
Minal は AI をファーストクラスで統合したターミナルエミュレータを目指す。

## 技術スタック

| レイヤー | 技術 |
|---------|------|
| 言語 | Rust |
| GPU レンダリング | wgpu |
| ウィンドウ管理 | winit |
| フォントレンダリング | cosmic-text |
| ターミナルエミュレーション | 自作 (VT100/VT220/xterm 互換) |
| PTY | portable-pty |
| テキストシェーピング | rustybuzz |
| macOS 統合 | objc2, cocoa crates |
| AI バックエンド | reqwest (HTTP), tokio (async) |
| 設定 | TOML (toml crate) |

## AI 機能詳細

### 1. AI コマンド補完

- シェル入力中にリアルタイムでコマンド候補をオーバーレイ表示
- 現在のディレクトリ、直近のコマンド履歴、プロジェクト構造をコンテキストとして利用
- Tab でサジェストを受け入れ、Esc で無視
- ローカルモデル優先（低レイテンシ）、フォールバックでクラウド API

### 2. インライン AI チャット

- `Ctrl+Shift+A` でターミナル下部にチャットパネルを開く
- ターミナルの現在の出力・コンテキストをAIに送信
- AIの回答からコマンドをワンクリックで実行可能
- マルチターン会話対応

### 3. セッション分析

- ターミナル出力をバックグラウンドで監視
- エラー検出時に自動で通知・修正案を表示
- ビルドエラー、テスト失敗、コマンドエラーを自動分類
- `Ctrl+Shift+E` でエラーサマリーパネル表示

### 4. スマートコンテキスト

- Git ブランチ、変更ファイル、最近のコミットを自動認識
- プロジェクトの言語・フレームワークを自動検出
- AI への問い合わせ時にこれらのコンテキストを自動付与

## 実装フェーズ

### Phase 1: 基盤ターミナル (MVP)
1. winit でウィンドウ作成、wgpu 初期化
2. PTY 生成とシェル起動
3. VT パーサー実装（基本的な ANSI エスケープシーケンス）
4. セルグリッドとスクロールバック
5. cosmic-text によるテキストレンダリング
6. 基本的なキー入力処理
7. 設定ファイル読み込み (TOML)

### Phase 2: ターミナル機能充実
1. 256色・TrueColor 対応
2. マウスイベント対応
3. クリップボード統合
4. タブ / 分割ペイン
5. フォント設定・テーマ
6. macOS ネイティブメニューバー統合

### Phase 3: AI 統合
1. AI プロバイダー抽象化レイヤー
2. コンテキスト収集エンジン
3. AI コマンド補完
4. インラインチャットパネル UI
5. セッション分析エンジン

### Phase 4: 磨き込み
1. パフォーマンス最適化
2. アクセシビリティ
3. プラグインシステム
4. Homebrew での配布
